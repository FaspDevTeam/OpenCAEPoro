bool ReadLine ( ifstream &  ifs,
vector< string > &  result

ReadLine is the core function while inputting the file. It will capture the next line which is meanningful, for example, not blank line or comments, and then gets rid of some useless characters such as space, commas. Finally, the segments of rest string will be stored in result. And if return false, it indicates we have reach the end of file.
